# Waffle Interface

### Experimental Environments
- Ubuntu 20.04.6 LTS (5.15.0-72-generic)
- Intel(R) Core(TM) i9-9900K CPU @ 3.60GHz
- 64 GB main memory
- 500 GB NVMe SSD

### How to Execute
1. In src/App.js, set the IP address where Waffle is running  
2. npm start
